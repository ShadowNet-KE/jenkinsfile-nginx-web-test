# Jenkinsfile nginx web test

## Requirements for Nodo

- Update Ubuntu:

`sudo apt-get update`

- Install Java:

`sudo apt install openjdk-8-jre-headless`

- Install Git

`sudo apt install git`

- Install Docker:

`sudo apt install docker.io`

- Added permission for all users

`sudo chmod 666 /var/run/docker.sock`
